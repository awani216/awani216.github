Research Project on Fuel Flow Optimization

Indian Institute of Technology Kanpur | Mentor: Faiz Hamid

I did a winter research internship under prof Faiz Hamid of IIT Kanpur. The internship focused on Data Mining and Machine Learning. It was divided into two phases. The first phase was to predict the rate of fuel consumption by employing multiple machine learning techniques. The second phase consisted of designing a recommendation system which will recommend optimal values of flight parameters to minimize fuel consumption. In the first phase of the project, I started by cleaning the data. It consisted of removing noise and detecting erroneous data points and replacing it with a relevant measure of central tendency. There were more than 180 variables, to begin with, and by employing tools like correlation and Random Forest Importance, I reduced the number of variables to 39. For prediction purposes, I used prediction models like random forest classifiers, Support Vector Machines etc. I was able to achieve an accuracy of 94% with Random Forest Classifier. The next phase was designing a recommender system. I proposed an algorithm which used K Nearest Neighbours approach to find out conditions in the dataset which were similar to the flight conditions specified. This was followed by implementation of Random Forest importance and Correlation to learn about the dependence of variables on the fuel flow subjected to these conditions. Based on the results of these tests, the parameters were adjusted to minimize the fuel flow. I was able to reduce the fuel consumption value by a margin of 3-4%. Currently, Prof Faiz Hamid is working for the publication of this research.

Remote Data Analytis Internship

Indian Institute of Management Lucknow

The purpose of this project is to analyze the pricing strategy of hotels in the Indian hotel industry. Many factors drive hotel room prices. The objective of this project is to identify the factors that matter the most. After analyzing the data provided by www.hotel.in, I concluded that we can consider room rent as a function of date, hotel features which include facilities that hotel provides like AC, wifi, breakfast, etc. and external factors which include the location of the hotel, proximity to tourist destinations etc. I was also able to predict the prices. In this project, I learned regression analysis, chi-square test, etc.
